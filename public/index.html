Garmin Gallery Extractor

Generá una sola carpeta con todas las imágenes del lote.

Funciona con URLs completas o con SKUs sueltos. Si ingresás solo el SKU, la app arma automáticamente la URL usando el formato de Garmin y guarda todo junto en una carpeta única.

Entrada flexible

Podés mezclar URLs y SKUs en el mismo envío.

Salida cloud-friendly

En local se arma una sola carpeta con todas las imágenes juntas.

Compatible con Cloud Run

No depende de carpetas persistentes dentro del servidor.

Nueva extracción

Preparar descarga

SKU https://www.garmin.com/es-MX/p/0000000/pn/<SKU>/

Formato de archivo: SKU_1.jpg

Cómo se comporta en producción

Qué pasa en Cloud Run

  • En local, los archivos quedan en una sola carpeta dentro de downloads/.
  • En Cloud Run el filesystem sigue siendo temporal.
  • Si querés persistencia en la nube, lo correcto sigue siendo usar Cloud Storage.
