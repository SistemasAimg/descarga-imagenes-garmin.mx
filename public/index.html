Garmin Gallery Extractor

Descargá el carrusel principal como un ZIP ordenado por SKU.

Funciona con URLs completas o con SKUs sueltos. Si ingresás solo el SKU, la app arma automáticamente la URL usando el formato de Garmin.

Entrada flexible

Podés mezclar URLs y SKUs en el mismo envío.

Salida cloud-friendly

El usuario descarga un ZIP local con carpetas por SKU.

Compatible con Cloud Run

No depende de carpetas persistentes dentro del servidor.

Nueva extracción

Preparar descarga

SKU https://www.garmin.com/es-MX/p/0000000/pn/<SKU>/

El ZIP mantiene esta estructura: SKU/imagen-01.jpg

Cómo se comporta en producción

Qué pasa en Cloud Run

  • La app genera el ZIP y el navegador del usuario lo descarga.
  • No conviene guardar carpetas permanentes dentro de Cloud Run.
  • Si más adelante querés historial, lo correcto es sumar Cloud Storage.
